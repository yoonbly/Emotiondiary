# ✍ 감정일기장 클론코딩
## why?
- 스터디에서 토이 프로젝트를 수행하기 위해 react언어에 익숙해지기위해 해당 강의를 수강 후 프로젝트를 진행하였습니다.   
- 프로젝트 수행 결과, react component/hook/props에 대한 전반적인 이해, react프로젝트를 처음부터 진행하는 과정을 배울 수 있었습니다.
